ROAD

Game Design Document

By: Jerry Ding, Ryan Chung, Rocco Persico

INTRODUCTION

This document describes a game called "ROAD", which is designed to be a bullet hell game with rogue-like elements. The game will use sprite-based animation, collision detection, AI, screen scrolling, and more 2D game techniques.

TECHNOLOGY

ROAD will be developed for the Windows Platform using the Wolfie2D game engine, which is a bare-bones engine developed by Richard McKenna and Joe Weaver for rapid prototyping 2D games. Artwork will mostly consist of pixel art, so we'll use Piskel and MS Paint.

BACK STORY

Max is a lone wanderer in a post-apocalyptic world, or at least he became alone the day the Scorpion Gang ruthlessly murdered his wife. Max was devastated and sought revenge against the Scorpions. Max used his prior tinkering skills to create a vehicle capable of fueling his revenge. Until Max successfully enacts his revenge, he continues to wander the lonely ROAD.

OBJECTIVE

Max’s task is to defeat all the leaders of the Scorpion Gang. Max traverses the ROAD in his custom-built vehicle in search of these leaders. Along his search, Max finds various scrap parts that can be used to upgrade his vehicle. Max also finds pit-stops along the way which allow him to use his scrap parts in order to upgrade his vehicle or trade. As Max progresses, the Scorpion Gang will become more aware of Max’s presence and unleashes stronger gang members to take Max down. Max must defeat a leader before moving into a new region.

GAMEPLAY

The game will play from the top-down perspective, with Max driving upwards towards the end of the level. Max is able to turn left and right, and drive forwards and in reverse as long as he stays inside the viewport that is constantly moving up.

Throughout the levels and stages, Max will encounter a variety of different enemies. Each region will feature a unique set of enemies to fight. Some examples of enemies include:

Name

Description

Biker

Found in region 1. Bikers are small, move fast, and fire slowly, but are not very accurate.

Trucker

Found in region 2. Truckers are large, move slowly, and fire moderately fast, and are moderately accurate. Truckers will also attempt to ram into the player.

Tank

Found in region 3. Tanks are very large and move very slowly. Tanks have two weapons: a fast firing machine gun and a powerful missile launcher. Tanks will also attempt to run over the player.

After clearing each level, Max will be able to buy stat upgrades with his spare parts. The stats available for the player to upgrade are as follows:

CONTROLS

This game will be played using both a keyboard and mouse. Once started, use the following:

  • W - Move Up
  • A - Move Left
  • S - Move Down
  • D - Move Right
  • SPACE - Use Item
  • MOUSE MOVE - Aim weapon
  • LEFT MOUSE CLICK - Use weapon
  • SCROLL WHEEL - Cycle weapons
  • ESC - If game in progress, pauses game and brings up menu. Otherwise, does nothing

GRAPHICAL USER INTERFACE

The GUI will be as follows:

  • Splash Screen - The splash screen GUI simply presents a game logo and a Start button for the user to press when they are ready to play. If pressed, the game then presents the main menu screen.
  • Main Menu - Before a game is started, we show the main menu so the player can choose from the following options:
  • New Game - Start a new game.
  • Level Select - Allows the player to select the level(for testing purposes)
  • Controls - Display an info screen that describes all game controls.
  • Help - Displays an info screen that tells the backstory. Also serves as a cheat menu(for testing purposes)
  • In-Game Menu - While the game is in progress, we'll always have a game menu that allows the player to select from the following options:
  • Resume - Closes the in-game menu
  • Controls - Displays the in-game controls
  • Exit - Ends the current game and returns to the main menu
  • In-Game GUI - While playing the following will be displayed at all times:
  • Scrap Parts Collected
  • Vehicle Health
  • Note: during boss fights, the total remaining health of the boss will also be displayed on-screen.

ARTWORK

All artwork in the game will be original. The following needs to be created:

  • Max- we'll only see Max in his vehicle, so we'll need to make artwork for the vehicle. It will require animations for:
  • Moving Right
  • Moving Left
  • Moving Up
  • Moving Down
  • Collide With Enemy at high speed
  • Taking Damage
  • Dying
  • Using Weapons

SOUND EFFECTS

All sound effects will be original. Sounds must be made to coincide with each of the following events:

  • Driving

  • Colliding with Enemies

  • Vehicles Exploding

  • Picking Up Power-Ups
  • Firing Weapons

  • Max Celebrating Clearing a Level
