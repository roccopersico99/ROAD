CSE 380 Final Project
ROAD by Rocco Persico, Ryan Chung, and Jerry Ding

The source code for the final version of ROAD can be accessed in the benchmark3 folder
Inside the benchmark3 folder, there will be a dist folder with an index.html inside
To locally host ROAD, direct Web Server for Chrome to dist/index.html (not src/index.html)
To run the game, cd to public/benchmark3 and run the 'gulp' command

Github Repo Link: https://github.com/roccopersico99/ROAD
